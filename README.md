#  Spinix Sales Analysis

##  Project Overview
This project analyzes sales performance for **Spinix**, focusing on revenue trends, product performance, customer behavior, and business KPIs.  
The goal is to provide clear insights that support decision-making in pricing, marketing, and inventory planning.

This analysis was done using **Excel** (data cleaning & exploration) and **Power BI** (visualization & dashboard building).

---

##  Key Objectives
- Analyze overall sales performance  
- Identify top and low-performing products  
- Study customer buying behavior  
- Explore revenue trends over time  
- Build a business-ready Power BI dashboard  



## Tools & Technologies
- **Excel**  
  - Data cleaning  
  - Pivot tables  
  - Preliminary analysis  
- **Power BI**  
  - Data modeling  
   
  - Interactive dashboard  



##  Key Insights
- Identified top-performing product categories  
- Highlighted underperforming items with low sales contribution  
- Month-to-month revenue changes revealed seasonal patterns  
- Customer segments and purchase frequency helped show retention trends
